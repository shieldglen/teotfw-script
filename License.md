## How to Install 📥

### Standard Installation (Windows)

1. Download the file from the GitHub link below
2. Run the executable (no installation required — I designed it to be portable)
3. Select season (1 or 2) from the dropdown
4. Choose episode number (1-8)
5. Read the full script in the main window
6. Use the search bar to find specific quotes
7. Bookmark your favorites for later

## Troubleshooting 🔧

| Issue | Solution |
|-------|----------|
| **Script not loading** | Restart the app, select episode again |
| **Search not working** | Check spelling, try a shorter word |
| **Bookmarks not saving** | Run as Administrator, check folder permissions |
| **App crashes** | Update Windows, re-download the file |
| **Windows Defender blocks** | Add to exclusions — the app is safe, I promise |
